#Pyhon Summary
1) У любого обькта в языке python есть, идентификатор тип и значение
	- Идентификатор позволяет в любой момент времени отличить один обьект от другого, потому что гарантируется, что никакие два обьекта в один момент времени не обладают одинаковым идентификатором
	- Идентификатор обьекта не меняется с самого его созданиея 
2) Переменная в языке Python - это всего лишь ссылка на обьект, значение переменной - это всегда значение обьекта, если
	несколько переменных ссылаются на один и тот же обьект, то их значения всегда будут совпадать
