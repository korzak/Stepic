При записи сложных выражений надо иметь в виду приоритеты операторов. В следующем списке приведены приоритеты всех имеющихся в Java операторов. Чем выше позиция в списке, тем выше приоритет оператора, т.е. тем раньше он применяется. Чтобы не запутаться, рекомендуется использовать скобки.

1)постфиксные: expr++ expr--
2)﻿унарные: ++expr --expr +expr -expr ~ !
3)мультипликативные: * / %
4)аддитивные: + -
5)сдвиги: << >> >>>
6)сравнения: < > <= >= instanceof
7)равенство: == !=
8)побитовое И: &
9)побитовое исключительное ИЛИ: ^
10)побитовое ИЛИ: |
11)логическое И: &&
12)логическое ИЛИ: ||
13)тернарный: ? :
14)присваивания: = += -= *= /= %= &= ^= |= <<= >>= >>>=
